SrssFileExts.imageExtensions¶

File extensions referring to images.

>>> from shinerainsevenlib.standard import *
>>> # (this will import files and SrssFileExts)
>>>
>>> path = '/path/to/file.jpg'
>>> if files.getExt(path) in SrssFileExts.imageExtensions:
>>>     print('looks like an image')

SrssFileExts.extensionPossiblyExecutable(s)¶

Returns ‘exe’ if it looks executable,.

Returns ‘warn’ if it is a document type that can include embedded scripts,

Returns False otherwise




SrssFileExts.isCompressedTarExtension(path)¶

Is this a compressed tar archive, like example.tar.gz or example.tgz’ Not example.gz which should fall under isSingleFileCompressionExtension() instead.




SrssFileExts.isSingleFileCompressionExtension(path)¶

Is this a plain single-file-archive, like example.gz’ Not example.tar.gz which should fall under isCompressedTarExtension() instead.
