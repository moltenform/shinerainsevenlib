.. py:data:: imageExtensions

   File extensions referring to images

   >>> from shinerainsevenlib.standard import *
   >>> # (this will import files and SrssFileExts)
   >>> 
   >>> path = '/path/to/file.jpg'
   >>> if files.getExt(path) in SrssFileExts.imageExtensions:
   >>>     print('looks like an image')

.. py:function:: extensionPossiblyExecutable(s)

   Returns 'exe' if it looks executable,

   Returns 'warn' if it is a document type that can include embedded scripts,

   Returns False otherwise
